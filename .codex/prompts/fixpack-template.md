# FIXPACK Template (Post NO-GO)

## Назначение
Небольшой пакет исправлений после NO-GO.

## Правила
- Исправлять только пункты из `Must fix`.
- Без дополнительных рефакторингов.
- Scope и пути те же, что в исходном workpack (если не переутверждено).

## Verification
- Запускать только targeted verification, покрывающую Must fix.
- Результаты верификации указывать явно по каждой проблеме.

## Output
1. Fixed items
2. Files consulted
3. Files changed
4. Commands run
5. Remaining risks
