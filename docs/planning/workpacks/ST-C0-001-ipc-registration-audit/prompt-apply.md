# ST-C0-001 Prompt — APPLY (Template, blocked until Gate)

⚠️ Этот prompt можно использовать только после:
1) Human Gate approval;
2) утверждённого PLAN с file-level шагами.

РЕЖИМ: APPLY.

Инструкция:
1. Выполняй только утверждённый PLAN.
2. Соблюдай allowed/forbidden paths из `workpack.md`.
3. Делай минимальный diff.
4. Stop-the-line при любом отклонении от плана.
5. Запускай только согласованные verification команды.
6. В отчёте обязательно укажи: files consulted, files changed, commands run, risks.

Запрещено:
- начинать APPLY без Gate;
- расширять scope;
- добавлять зависимости.
