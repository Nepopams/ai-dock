# Definition of Done (DoD)

Изменение считается завершённым, если:

1. Diff соответствует утверждённому workpack.
2. Изменения минимальны и не содержат лишнего scope.
3. Runtime-код не изменён вне разрешённых путей.
4. Обновлены docs/indexes/contracts, если это требуется scope.
5. Тесты/сборка запущены **или** в отчёте указана причина невозможности.
6. Нет утечки секретов в коде/доках/логах.
7. Не добавлены новые зависимости без обоснования и согласования.
8. Review Gate пройден (A/B/C/D согласно governance).
